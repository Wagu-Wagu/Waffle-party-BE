## ❤️ 기획 배경
**와플파티 - OTT 같이 볼 사람이 필요할 때** <br> 

OTT를 같이 보면서 채팅하는 `왓챠파티`, `Teleparty`, `Screena`에서 **함께 볼 사람을 모집**하는 요구사항에서 기획을 하게 되었습니다. <br> 

기존의 '다음카페', '트위터' 등의 특정 커뮤니티에서 함께 볼 사람을 모집하는 어려움을, '와플파티'를 통해서 함께 같이 볼 사람을 쉽게 구할 수 있습니다. <br>
